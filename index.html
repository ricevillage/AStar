About A* search

 A* is an informed search algorithm, or a best-first search. Specifically, A* selects the path that minimizes f(n), the number on the cells.

f(n) = g(n) + h(n), where:

1. n is the next node on the path.

2. g(n) is the cost of the path from the start node to n.

3. h(n) is a heuristic function that estimates the cost of the cheapest path from n to the goal; we use Euclidean distance.
